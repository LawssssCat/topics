groups:
  - title: 'Website goals'
    items:
      - 'Articulated goals accurately and succinctly '
      - 'Described what users want to accomplish'
      - 'Wrote Accurate value proposition'
      - 'Compared and contrasted appropriate competitors'

  - title: 'Personas'
    items:
      - 'Researched thoroughly the target audience'
      - 'Created the appropriate number of detailed personas'
      - 'Documented core user desires & drivers'
      - 'Documented core user pain points'

  - title: 'Task flows'
    items:
      - 'Wrote user stories that describe real usage scenarios'
      - 'Created descriptive task flows to orient screens'
      - 'Designed the information architecture & the sitemap'
      - 'Simplified task flows to the minimum number of screens & actions'

  - title: 'Wireframes'
    items:
      - 'Wireframed every significant screen'
      - 'Tested and adjusted wireframes with real user feedback'
      - 'Considered user flows through wireframes and information architecture'
      - 'Accounted for hierarchy in wireframe designs'
      - 'Confirmed primary actions are always visible'

  - title: 'Visual design'
    items:
      - 'Created a cohesive brand'
      - 'Designed appropriate logos for all sizes: large, `64px`, `32px`, `16px`'
      - 'Created mockups of buttons'
      - 'Designed some of the typesetting'
      - 'Created mockups of important patterns: banners, cards, etc.'
      - 'Considered colour contrast and other accessibility concerns'
      - 'Accounted for download times and performance'
      - 'Considered animations, transitions & motion'
      - 'Got feedback from real users'

  - title: 'Micro copy'
    items:
      - 'Considered the language: formal? informal? slang?'
      - 'Determined a few button texts'
      - 'Wrote some call to actions'
      - 'Prepared some descriptions, navigation names, etc.'
      - 'Wrote error & confirmation message words'
      - 'Considered empty state text'
